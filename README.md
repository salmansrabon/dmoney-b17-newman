# Project Title: Dmoney user management
## Project description: Dmoney is a fake money transaction portal where user can send money, deposit, cashout and payment to other users.

### Technology used:
- NodeJS
- Postman
- Newman
- HTML Report Extra

### How to run:
- `npm i`
- `node report.js`
